# Database

## User
- id (BigInt)
- email (varchar(255))
- password (hash) (BigInt)
- OPTIONAL name (varchar(255)) 

## Product
- id (BigInt)
- name (varchar(255))
- price (int)
- OPTIONAL amount (int) 

## Buying
- id (BigInt)
- user_id (BigInt, fk)
- OPTIONAL date_and_time_of_buying (DATETIME) 

## Product_in_basket suggestion to rename Bought_product
